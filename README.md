# 📊 HR Analytics Dashboard

## 📌 Project Overview

This project presents an interactive HR Analytics Dashboard developed using Microsoft Power BI to analyze employee attrition, workforce distribution, salary trends, and other key HR metrics.

---

## 🎯 Objective

The objective of this project is to help HR departments identify factors affecting employee attrition and support data-driven workforce decisions.

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📈 Key KPIs

- Total Employees
- Attrition Rate
- Average Age
- Average Salary
- Active Employees
- Department-wise Attrition

## 💡 Key Insights

- Highest attrition observed in specific departments.
- Employees working overtime showed higher attrition.
- Young employees had comparatively higher resignation rates.
- Salary and job role influenced employee retention.
